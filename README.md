## 🚀 Featured Project

### Real-Time Gaze Tracking System

A real-time gaze tracking system that estimates **horizontal gaze direction (Left / Center / Right)** using MediaPipe's Face Landmarker Tasks API.

The system uses **iris and eye corner landmarks** to calculate normalized iris positioning and determine gaze direction.

Key features:

* Landmark-based gaze estimation using MediaPipe
* Real-time video processing with OpenCV
* Exponential smoothing for noise reduction
* Calibrated thresholds for stable gaze classification
* Migrated from Haar Cascade detection to a geometric landmark approach for better robustness

Technologies used:

* Python
* MediaPipe
* OpenCV
* Computer Vision
* Signal Processing
