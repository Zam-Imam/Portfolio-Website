01 //

Real-Time Gaze Tracking System

A real-time gaze tracking system that estimates horizontal gaze direction (Left / Center / Right) using MediaPipe's Face Landmarker Tasks API. Built on iris and eye corner landmark detection with normalized iris positioning, exponential smoothing for noise filtering, and calibrated thresholds for stable classification. Migrated from Haar Cascade to a geometric landmark-based approach for better lighting robustness.
